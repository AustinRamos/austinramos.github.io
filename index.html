What is this?

A way to show off some of the cool things i've worked on, but more importantly, a way to hold myself accountable for learning new things! I like teaching myself new stuff, but I tend to bounce around 4-5 topics that fascinate me at any given time, without learning about any one in particular in depth. This is just no way to learn much about anything. Maybe this is something most people already know, but I have the attention span of a spoon so this is a big step for me. Hopefully posting projects on here will help me stay focused.

I’ll break down some projects I’ve done in the past, write about the technical interview process for software engineering these days, write about topics that are loosely related to my interests below, or pretty much whatever else I want to!
